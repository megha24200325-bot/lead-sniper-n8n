# lead-sniper-n8n
This project is an automated High-Value Lead Tracker built using n8n, GitHub API, Groq AI, and Discord Webhooks. The workflow fetches users who starred a GitHub repository, processes them one by one, and retrieves detailed profile information such as name, bio, company, followers, and repositories. It then filters high-value users based on conditions like more than 100 followers or 50 public repositories. For the selected users, an AI model generates a personalized one-line sales pitch, and the final output is sent to a Discord channel using a webhook. This project demonstrates how automation and AI can be combined to identify and engage potential leads efficiently, while also handling API rate limits and maintaining security by removing API keys from the shared workflow.
